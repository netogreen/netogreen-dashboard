🚨 1. 최연식 사원 — Drive 소유 2건(최하위) but 수정 217회 · "운송장 변환" 69회 반복
문서 소유는 2건으로 전체 최하위이나, 실제 수정 횟수는 217회로 적지 않음. 핵심 패턴: "운송장 변환(대한,우체국,화성2,소싱)" 시트를 69회 반복 수정하며, 프로모션(29회), 회원세그먼트_CRM(26회), 고객 전달용 발송내역(19회)까지 Top 3 집중도 57%.
→ 액션: 운송장 변환 작업은 자동화 가능성 높음 — Apps Script 또는 RPA로 반복 입력 제거. 단순 수정 업무에서 벗어나 Biz. 분석 역할로 전환 가능.
⚠️ 2. 이희선 사원 — 농장별 재배환경관리 1인 136회 집중 (Top 3 집중도 80%)
Notion 0건이나, Drive 수정 245회 중 "농장별 재배환경관리_V1"에만 136회(56%) 집중. 대체 품종 검토(33회), 화성4 New(26회)까지 Top 3에 전체 수정의 80%가 몰림. Lab. 위클리 참여는 확인됨.
→ 액션: ① 재배환경관리 시트의 반복 입력 패턴 분석 → 센서 데이터 자동 입력 연동 검토 ② 백업 담당자 지정(1인 의존 리스크) ③ Notion 병행 안내.
⚠️ 3. CEO 역할 전환 — 소유 108건(2위) but 수정 147회(최하위)
임형준 대표의 Drive 소유 파일은 108건(2위)이나, 실제 수정 횟수는 147회로 전 직원 중 최하위. Top 3가 인사(26회), 조직도(15회), 네토그린_출자검토(11회)로 경영·인사·투자 검토에 집중. 이전 분석 대비 농장 실무 시트 수정이 크게 줄었음.
→ 해석: CEO가 실무에서 경영 업무로 전환 중인 긍정적 신호. 다만 채널톡 376건(42%)은 여전히 높아 커뮤니케이션 위임 필요.
📊 4. 신용재 이사 — 수정 749회(1위) · 농장 관리 + 정산 핵심 허브
실제 편집 기준 전 직원 1위(749회). 농장주 정보(92회), 박스 사용량 취합(39회), 농장별 New 시트 다수(고양2 37회, 진안1 34회, 수원1 34회). 금주에도 프리셀 정산서 10건 이상 반복 수정.
→ 액션: 프리셀 정산서 반복 생성은 자동화 가능성 높음 — 정산 템플릿 + Apps Script 자동 생성. 산도조절제 요청도 Google Forms 연동 검토.
📊 5. 최종윤 과장 — 수정 724회(2위) · 농장 데이터 전담 (Top 3 집중도 29%)
재배유닛 현황(109회), 고양3 New(52회), 고양1 New(46회), 평택1(44회), 고양4(44회) 등 농장별 시트를 고르게 관리. Top 3 집중도 29%로 가장 낮아 업무가 다양하게 분산됨.
→ 액션: Notion 활용 재개 유도 + 재배ERP 시스템과 시트 데이터 연동 자동화 검토.
✅ 6. 손소희 대리 — 수요&공급 124회 + Biz. 운영 핵심 (462회)
"수요&공급, 마케팅비용" 시트 124회 수정으로 Biz. 팀 운영 1위. 소싱상품 단가분석(43회), 프로모션(39회), 프리셀 소싱상품 판매기록(33회)까지 폭넓게 관리.
→ 액션: 손소희-최연식 업무 분담 재설계. 소싱상품 단가 분석 등 반복 업무 자동화 검토.
✅ 7. 상승규 팀장 — COGS 150회(전체 1위 문서) + 발주·재고 관리
COGS 시트를 150회 수정으로 개인-문서 단위 전체 1위. 발주서_주연하이테크(51회), 입출고&재고관리(44회)까지 Engineer 핵심 프로세스 전담.
→ 액션: COGS·발주·재고 3종 시트의 데이터 연동 자동화 검토. 견적서 템플릿 자동 생성도 병행.
